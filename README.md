О программе:

Программа представляет собой автоматический компоновщик изображений для печати. Её главная цель — взять набор изображений, загруженных пользователем, и разместить их на листах формата А4 максимально компактно, чтобы сэкономить бумагу при распечатке, а затем выдать готовый файл Microsoft Word (.docx).

Основные функции:

1. Загрузка изображений:
    - Поддержка Drag-and-Drop и выбора файлов через диалоговое окно.
    - Принимает любые форматы (JPG, PNG, GIF и т.д.).
    - Есть возможность очистить макет, удалив все картинки

2. Умная авто-компоновка (Layout Engine):
    - Это сердце программы. Алгоритм не просто ставит картинки в ряд. Он пытается найти идеальное место для каждой картинки, чтобы заполнить пустоты.
    - Поворот: Если картинка не влезает горизонтально, но влезает вертикально (или наоборот), программа автоматически повернет её на 90 градусов.
    - Приоритет: Алгоритм настроен так, чтобы оставлять пустое место внизу страницы (для заметок или экономии бумаги), прижимая контент к верхнему левому или правому краю.

3. Магия фона: Одной кнопкой удаляет белый фон и обрезает лишние пустые края у картинки.

4. Предпросмотр (Preview):
	- Визуальное отображение листов А4 с учетом реальных отступов.
	- Показывает, как именно будут расположены картинки.
	- Отображает точные размеры каждой картинки при наведении.

5. Экспорт в Word (.docx):
    - Генерация реального документа Word.
    - Использует "плавающую" (absolute) верстку внутри Word, что гарантирует, что картинки не съедут при открытии файла.
    - Учитывает поворот изображений (физически поворачивает байты картинки перед вставкой в документ).
